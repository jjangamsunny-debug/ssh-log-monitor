# SSH Log Monitoring System

This project monitors SSH login attempts on a Linux server and stores them in a MySQL database.

## Features

- Detect successful SSH login
- Detect failed SSH login
- Store logs in MySQL
- Real-time log monitoring

## Technologies Used

- C++
- MySQL
- Linux

## Logs Location

Ubuntu/Debian:

/var/log/auth.log

CentOS/RHEL:

/var/log/secure
